ПРИЛОЖЕНИЕ 1. Трекер личных расходов
=====================================

Что делает:
- Добавление расхода: сумма + категория (Spinner) + дата.
  Дата выбирается через системный DatePickerDialog (календарь):
  по нажатию на кнопку "Выбрать дату" открывается календарь, выбранная
  дата подставляется на кнопку и сохраняется вместе с расходом.
- Список всех трат (ListView + ArrayAdapter).
- Итоговый баланс за выбранный месяц (Spinner с 12 месяцами).
- Данные сохраняются в SharedPreferences как JSON-строка (org.json,
  встроен в Android SDK, это НЕ сторонняя библиотека).

3) AndroidManifest.xml
   -> app/src/main/AndroidManifest.xml
   Можно заменить целиком ИЛИ просто сверить <activity> и label.

--------------------------------------------------
ЧТО ПОМЕНЯТЬ РУКАМИ
--------------------------------------------------

Android Studio при создании проекта генерирует свой package
(например com.example.myapp). Нужно привести имя пакета в соответствие:

- Первая строка MainActivity.java:
      package com.example.expensetracker;
  замените expensetracker на реальный пакет вашего проекта.

- В AndroidManifest.xml атрибут package="com.example.expensetracker"
  (если используете чистый Android без namespace в build.gradle).
  В новых проектах package задаётся в build.gradle (namespace) — тогда
  атрибут package в манифесте можно убрать, главное чтобы MainActivity.java
  лежал в правильной папке пакета.

PERMISSIONS
--------------------------------------------------
Не требуются (нет сети, нет файлов, нет камеры).

--------------------------------------------------
НЮАНСЫ
--------------------------------------------------
- Тема @style/Theme.AppCompat.Light.NoActionBar — если проект без
  AppCompat, поставьте свою тему или @android:style/Theme.Material.Light.
- Дата выбирается через DatePickerDialog. По умолчанию стоит сегодняшняя
  дата; спец. разрешений для календаря не требуется.
- Баланс считается по номеру месяца (0-11), год не учитывается —
  для учебного приложения этого достаточно.
- minSdk: подойдёт любой (21+), спец. API не используется.
