ПРИЛОЖЕНИЕ 4. Список дел
=====================================

Что делает:
- Добавление новых задач.
- Короткий тап по задаче — отметить выполненной (текст зачёркивается)
  или снять отметку.
- Долгое нажатие по задаче — удалить её.
- Список сохраняется между запусками (SharedPreferences, JSON через org.json).

Реализация: задачи хранятся как JSONArray объектов {text, done}.
Свой адаптер TaskAdapter (наследник BaseAdapter) рисует текст и включает
флаг Paint.STRIKE_THRU_TEXT_FLAG для выполненных задач.

--------------------------------------------------
КУДА КОПИРОВАТЬ ФАЙЛЫ
--------------------------------------------------
1) MainActivity.java  -> app/src/main/java/com/example/<пакет>/MainActivity.java
2) activity_main.xml  -> app/src/main/res/layout/activity_main.xml
3) AndroidManifest.xml -> app/src/main/AndroidManifest.xml

--------------------------------------------------
ЧТО ПОМЕНЯТЬ РУКАМИ
--------------------------------------------------
- Первая строка MainActivity.java: package com.example.todolist;
  заменить на реальный пакет проекта.
- В AndroidManifest.xml package="com.example.todolist" привести
  в соответствие (или убрать, если namespace задан в build.gradle).

Не требуются.
