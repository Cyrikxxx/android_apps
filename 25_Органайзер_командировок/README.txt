ПРИЛОЖЕНИЕ 25. Органайзер для командировок
=====================================

Что делает:
- Планирование поездки: город, даты (DatePickerDialog), цель.
- При создании автоматически формируется список задач:
  забронировать отель, купить билеты, оформить командировочное
  удостоверение, собрать документы, цель поездки.
- Spinner выбирает поездку, ListView показывает её задачи.
- Тап по задаче — отметить выполненной (✓) / снять отметку (☐).
- Кнопка "Удалить" убирает выбранную поездку.
- Данные сохраняются (SharedPreferences, org.json; задачи хранятся
  вложенным массивом внутри поездки).

--------------------------------------------------
КУДА КОПИРОВАТЬ ФАЙЛЫ
--------------------------------------------------
1) MainActivity.java  -> app/src/main/java/com/example/<пакет>/MainActivity.java
2) activity_main.xml  -> app/src/main/res/layout/activity_main.xml
3) AndroidManifest.xml -> app/src/main/AndroidManifest.xml

--------------------------------------------------
ЧТО ПОМЕНЯТЬ РУКАМИ
--------------------------------------------------
- package com.example.trips в MainActivity.java и в манифесте
  привести к пакету вашего проекта.

--------------------------------------------------
PERMISSIONS
--------------------------------------------------
Не требуются.

--------------------------------------------------
НЮАНСЫ
--------------------------------------------------
- Задачи хранятся как вложенный JSON-массив внутри объекта поездки —
  хороший пример работы с org.json.
- minSdk 19+ (JSONArray.remove).
