ПРИЛОЖЕНИЕ 6. Мини-галерея
=====================================

Что делает:
- Показывает сетку из 6 картинок (GridView, 3 в ряд).
- По нажатию на картинку открывается второй экран (DetailActivity)
  с увеличенным изображением.
- На экране просмотра кнопки "Назад"/"Вперёд" листают картинки по кругу.

ВАЖНО про картинки:
В коде используются СИСТЕМНЫЕ картинки Android (android.R.drawable.*),
чтобы проект собирался сразу без добавления файлов.
Чтобы поставить свои изображения:
  1) положите файлы (например img1.jpg ... img6.jpg) в res/drawable;
  2) в MainActivity.java в массиве images замените элементы на
     R.drawable.img1, R.drawable.img2, ... R.drawable.img6.

--------------------------------------------------
КУДА КОПИРОВАТЬ ФАЙЛЫ
--------------------------------------------------
1) MainActivity.java   -> app/src/main/java/com/example/<пакет>/MainActivity.java
2) DetailActivity.java -> app/src/main/java/com/example/<пакет>/DetailActivity.java
3) activity_main.xml   -> app/src/main/res/layout/activity_main.xml
4) activity_detail.xml -> app/src/main/res/layout/activity_detail.xml
5) AndroidManifest.xml -> app/src/main/AndroidManifest.xml

--------------------------------------------------
ЧТО ПОМЕНЯТЬ РУКАМИ
--------------------------------------------------
- package com.example.gallery в ОБОИХ .java файлах и в манифесте
  привести к пакету вашего проекта.
- Оба класса (MainActivity и DetailActivity) должны лежать в одной
  папке пакета.

--------------------------------------------------
PERMISSIONS
--------------------------------------------------
Не требуются (картинки берутся из ресурсов, не из интернета/памяти).

--------------------------------------------------
НЮАНСЫ
--------------------------------------------------
- Не забудьте, что в манифесте зарегистрированы ДВА activity.
- Листание сделано кнопками (проще, чем ViewPager) — этого достаточно
  для учебного задания.
- minSdk любой (21+).
