ПРИЛОЖЕНИЕ 21. Ежедневник с приоритетами и напоминаниями
=====================================

Что делает:
- Добавление задачи: текст, приоритет (Высокий/Средний/Низкий — Spinner),
  время (TimePickerDialog).
- Список задач ОТСОРТИРОВАН по приоритету (сначала высокий).
- При добавлении показывается уведомление-напоминание.
- Долгое нажатие — удалить.
- Данные сохраняются (SharedPreferences, org.json).

--------------------------------------------------
КУДА КОПИРОВАТЬ ФАЙЛЫ
--------------------------------------------------
1) MainActivity.java  -> app/src/main/java/com/example/<пакет>/MainActivity.java
2) activity_main.xml  -> app/src/main/res/layout/activity_main.xml
3) AndroidManifest.xml -> app/src/main/AndroidManifest.xml

--------------------------------------------------
ЧТО ПОМЕНЯТЬ РУКАМИ
--------------------------------------------------
- package com.example.diary в MainActivity.java и в манифесте
  привести к пакету вашего проекта.

--------------------------------------------------
PERMISSIONS / УВЕДОМЛЕНИЯ
--------------------------------------------------
- POST_NOTIFICATIONS — нужно с Android 13 (объявлено + запрос в рантайме).
- С Android 8.0 уведомления идут через канал.

--------------------------------------------------
НЮАНСЫ
--------------------------------------------------
- Сортировка через Collections.sort с Comparator по полю priority.
- Уведомление приходит при добавлении (демо). Для показа строго в
  назначенное время нужен AlarmManager.
- minSdk рекомендуется 21+.
