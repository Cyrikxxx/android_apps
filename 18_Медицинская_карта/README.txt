ПРИЛОЖЕНИЕ 18. Медицинская карта (лекарства, аллергии)
=====================================

Что делает:
- Список лекарств: название, дозировка, время приёма.
- Список аллергий и хронических заболеваний.
- При добавлении лекарства показывается уведомление-напоминание о приёме.
- Долгое нажатие по элементу любого списка — удалить.
- Данные сохраняются (SharedPreferences: два JSON-массива через org.json).

--------------------------------------------------
КУДА КОПИРОВАТЬ ФАЙЛЫ
--------------------------------------------------
1) MainActivity.java  -> app/src/main/java/com/example/<пакет>/MainActivity.java
2) activity_main.xml  -> app/src/main/res/layout/activity_main.xml
3) AndroidManifest.xml -> app/src/main/AndroidManifest.xml

--------------------------------------------------
ЧТО ПОМЕНЯТЬ РУКАМИ
--------------------------------------------------
- package com.example.medcard в MainActivity.java и в манифесте
  привести к пакету вашего проекта.

--------------------------------------------------
PERMISSIONS / УВЕДОМЛЕНИЯ
--------------------------------------------------
- POST_NOTIFICATIONS — нужно с Android 13 (объявлено в манифесте и
  запрашивается в рантайме).
- С Android 8.0 уведомления идут через канал.

--------------------------------------------------
НЮАНСЫ
--------------------------------------------------
- Уведомление показывается при добавлении лекарства (демонстрация).
  Для напоминаний строго по расписанию нужен AlarmManager/WorkManager.
- minSdk рекомендуется 21+.
