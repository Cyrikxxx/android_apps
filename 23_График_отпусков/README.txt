ПРИЛОЖЕНИЕ 23. График отпусков для команды
=====================================

Что делает:
- Переключатель роли (Spinner): Сотрудник / Администратор.
- Сотрудник: вводит имя, выбирает даты начала и конца отпуска
  (DatePickerDialog) и подаёт заявку.
- Проверка пересечений: если на пересекающийся период уже есть 2 не
  отклонённые заявки — новая не принимается (максимум 2 человека).
- Администратор: видит все заявки; тап по заявке меняет статус по кругу
  (Ожидает -> Подтверждён -> Отклонён).
- Данные сохраняются (SharedPreferences, org.json).

--------------------------------------------------
КУДА КОПИРОВАТЬ ФАЙЛЫ
--------------------------------------------------
1) MainActivity.java  -> app/src/main/java/com/example/<пакет>/MainActivity.java
2) activity_main.xml  -> app/src/main/res/layout/activity_main.xml
3) AndroidManifest.xml -> app/src/main/AndroidManifest.xml

--------------------------------------------------
ЧТО ПОМЕНЯТЬ РУКАМИ
--------------------------------------------------
- package com.example.vacation в MainActivity.java и в манифесте
  привести к пакету вашего проекта.

--------------------------------------------------
PERMISSIONS
--------------------------------------------------
Не требуются.

--------------------------------------------------
НЮАНСЫ
--------------------------------------------------
- Пересечение периодов проверяется по правилу: начало1 <= конец2 И
  начало2 <= конец1.
- Полноценная календарная сетка со всеми заявками не рисуется (для
  простоты используется список); при желании можно добавить CalendarView.
- minSdk рекомендуется 21+.
