ПРИЛОЖЕНИЕ 15. Персональный тренировочный дневник
=====================================

Что делает:
- Запись тренировки: упражнение, подходы, повторения, вес (дата ставится сама).
- История всех записей (долгое нажатие — удалить).
- График изменения веса по всем записям (рисуется на Bitmap через Canvas,
  без сторонних chart-библиотек).
- Встроенный таймер отдыха между подходами (60 секунд, CountDownTimer).
- Данные сохраняются (SharedPreferences, org.json).

--------------------------------------------------
КУДА КОПИРОВАТЬ ФАЙЛЫ
--------------------------------------------------
1) MainActivity.java  -> app/src/main/java/com/example/<пакет>/MainActivity.java
2) activity_main.xml  -> app/src/main/res/layout/activity_main.xml
3) AndroidManifest.xml -> app/src/main/AndroidManifest.xml

--------------------------------------------------
ЧТО ПОМЕНЯТЬ РУКАМИ
--------------------------------------------------
- package com.example.workout в MainActivity.java и в манифесте
  привести к пакету вашего проекта.

--------------------------------------------------
PERMISSIONS
--------------------------------------------------
Не требуются.

--------------------------------------------------
НЮАНСЫ
--------------------------------------------------
- График рисуется своими руками на Bitmap (линия по значениям веса).
  Для полноценных графиков обычно берут MPAndroidChart, но это сторонняя
  библиотека — здесь по правилам её не используем.
- График появляется, когда записей минимум 2.
- minSdk 19+ (JSONArray.remove).
