ПРИЛОЖЕНИЕ 11. Магазин покупок (корзина товаров)
=====================================

Что делает:
- Экран со списком товаров (название, цена, кнопка "Купить").
- По кнопке "Купить" товар добавляется в корзину.
- Экран корзины: список добавленных товаров, долгое нажатие — удалить,
  внизу общая стоимость.
- Содержимое корзины сохраняется между запусками (SharedPreferences, org.json).

--------------------------------------------------
КУДА КОПИРОВАТЬ ФАЙЛЫ
--------------------------------------------------
1) MainActivity.java  -> app/src/main/java/com/example/<пакет>/MainActivity.java
2) CartActivity.java  -> app/src/main/java/com/example/<пакет>/CartActivity.java
3) activity_main.xml  -> app/src/main/res/layout/activity_main.xml
4) activity_cart.xml  -> app/src/main/res/layout/activity_cart.xml
5) item_product.xml   -> app/src/main/res/layout/item_product.xml
6) AndroidManifest.xml -> app/src/main/AndroidManifest.xml

--------------------------------------------------
ЧТО ПОМЕНЯТЬ РУКАМИ
--------------------------------------------------
- package com.example.shop в ОБОИХ .java файлах и в манифесте
  привести к пакету вашего проекта. Оба класса — в одной папке пакета.

--------------------------------------------------
PERMISSIONS
--------------------------------------------------
Не требуются.

--------------------------------------------------
НЮАНСЫ
--------------------------------------------------
- В манифесте зарегистрированы ДВА activity (MainActivity и CartActivity).
- item_product.xml — это макет ОДНОЙ строки списка товаров (нужен для
  кнопки "Купить" в каждой строке).
- Корзина и список товаров общаются через общий файл SharedPreferences "shop".
- JSONArray.remove(int) — minSdk 19+.
