ПРИЛОЖЕНИЕ 20. Персональный бюджет (семейный учёт)
=====================================

Что делает:
- Добавление транзакции: тип (Доход/Расход), категория, сумма.
- Показывает баланс (доходы минус расходы), суммы доходов и расходов.
- Круговая диаграмма расходов по категориям за текущий месяц
  (рисуется на Bitmap через Canvas.drawArc, без сторонних библиотек).
- Список транзакций (долгое нажатие — удалить).
- Данные сохраняются (SharedPreferences, org.json).

--------------------------------------------------
КУДА КОПИРОВАТЬ ФАЙЛЫ
--------------------------------------------------
1) MainActivity.java  -> app/src/main/java/com/example/<пакет>/MainActivity.java
2) activity_main.xml  -> app/src/main/res/layout/activity_main.xml
3) AndroidManifest.xml -> app/src/main/AndroidManifest.xml

--------------------------------------------------
ЧТО ПОМЕНЯТЬ РУКАМИ
--------------------------------------------------
- package com.example.budget в MainActivity.java и в манифесте
  привести к пакету вашего проекта.

PERMISSIONS
