ПРИЛОЖЕНИЕ 8. Персональный словарь иностранных слов
=====================================

Что делает:
- Добавление слова: слово + перевод + пример использования.
- Список всех слов (долгое нажатие — удалить).
- Режим карточек: показывается слово, кнопка "Перевернуть" открывает
  перевод и пример, кнопка "Следующая" листает карточки по кругу.
- Данные сохраняются локально (SharedPreferences, JSON через org.json).

--------------------------------------------------
КУДА КОПИРОВАТЬ ФАЙЛЫ
--------------------------------------------------
1) MainActivity.java  -> app/src/main/java/com/example/<пакет>/MainActivity.java
2) activity_main.xml  -> app/src/main/res/layout/activity_main.xml
3) AndroidManifest.xml -> app/src/main/AndroidManifest.xml

--------------------------------------------------
ЧТО ПОМЕНЯТЬ РУКАМИ
--------------------------------------------------
- package com.example.dictionary в MainActivity.java и в манифесте
  привести к пакету вашего проекта.

--------------------------------------------------
PERMISSIONS
--------------------------------------------------
Не требуются.

--------------------------------------------------
НЮАНСЫ
--------------------------------------------------
- Разметка обёрнута в ScrollView, чтобы форма, карточка и список
  помещались на маленьких экранах.
- JSONArray.remove(int) — minSdk 19+.
