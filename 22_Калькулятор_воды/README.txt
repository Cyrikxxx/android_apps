ПРИЛОЖЕНИЕ 22. Калькулятор нормы воды с учётом активности
=====================================

Что делает:
- Рассчитывает суточную норму воды: 30 мл на 1 кг веса + прибавка за
  уровень активности (низкая/средняя/высокая).
- Кнопка "+ стакан (250 мл)" отмечает выпитое; показывается прогресс
  (выпито из нормы, проценты).
- Статистика за последние 7 дней.
- При отметке стакана показывается напоминание пить воду.
- Данные сохраняются (SharedPreferences: норма и объём по дням).

--------------------------------------------------
КУДА КОПИРОВАТЬ ФАЙЛЫ
--------------------------------------------------
1) MainActivity.java  -> app/src/main/java/com/example/<пакет>/MainActivity.java
2) activity_main.xml  -> app/src/main/res/layout/activity_main.xml
3) AndroidManifest.xml -> app/src/main/AndroidManifest.xml

--------------------------------------------------
ЧТО ПОМЕНЯТЬ РУКАМИ
--------------------------------------------------
- package com.example.water в MainActivity.java и в манифесте
  привести к пакету вашего проекта.

--------------------------------------------------
PERMISSIONS / УВЕДОМЛЕНИЯ
--------------------------------------------------
- POST_NOTIFICATIONS — нужно с Android 13 (объявлено + запрос в рантайме).
- С Android 8.0 уведомления идут через канал.

--------------------------------------------------
НЮАНСЫ
--------------------------------------------------
- Возраст в поле вводится, но в базовой формуле не используется (можно
  усложнить формулу при желании).
- Напоминания "каждые 2 часа" в фоне требуют AlarmManager с повтором
  (setRepeating) или WorkManager — здесь для простоты уведомление
  показывается при отметке стакана.
- Объём по дням хранится в SharedPreferences по ключам "ml_дата".
- minSdk рекомендуется 21+.
