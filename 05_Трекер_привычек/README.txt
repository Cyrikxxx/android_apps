ПРИЛОЖЕНИЕ 5. Трекер привычек
=====================================

Что делает:
- Создание привычек ("пить воду", "читать книгу").
- Тап по привычке — отметить/снять выполнение за сегодня.
- Для каждой привычки показывается процент выполнения за неделю
  и наглядная шкала последних 7 дней (✓ — выполнено, · — нет).
- Долгое нажатие — удалить привычку.
- История и отметки сохраняются между запусками (SharedPreferences, org.json).

Упрощение календаря: вместо полноценной сетки CalendarView показывается
компактная шкала последних 7 дней прямо в строке списка — этого достаточно
для учебного приложения и проще в реализации. Даты выполнения хранятся
как массив строк "гггг-мм-дд" внутри каждой привычки.
