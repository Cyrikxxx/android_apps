ПРИЛОЖЕНИЕ 16. Расписание занятий (студент/преподаватель)
=====================================

Что делает:
- Добавление пары: день недели (Spinner), предмет, время, аудитория,
  преподаватель.
- Второй Spinner-фильтр выбирает день, и список показывает только пары
  этого дня (получается недельное расписание по дням).
- Долгое нажатие — удалить пару.
- Данные сохраняются (SharedPreferences, org.json).

--------------------------------------------------
КУДА КОПИРОВАТЬ ФАЙЛЫ
--------------------------------------------------
1) MainActivity.java  -> app/src/main/java/com/example/<пакет>/MainActivity.java
2) activity_main.xml  -> app/src/main/res/layout/activity_main.xml
3) AndroidManifest.xml -> app/src/main/AndroidManifest.xml

--------------------------------------------------
ЧТО ПОМЕНЯТЬ РУКАМИ
--------------------------------------------------
- package com.example.schedule в MainActivity.java и в манифесте
  привести к пакету вашего проекта.
